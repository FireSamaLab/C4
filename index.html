Estimateur C4 Construction (MVP)

Cette application est servie par Node/Express à partir du dossier public.

Démarrez le serveur, puis ouvrez : http://localhost:3000

Si vous parcourez les fichiers directement, ouvrez : public/index.html

Note : Node.js est requis pour exécuter les API backend.

Installez Node.js, puis exécutez npm install et npm start.
